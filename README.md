# 弹幕时域密度分析系统 (多尺度感知增强版)

> Danmaku Time-Domain Density Analysis System

本项目旨在探索基于纯粹弹幕密度的时域分析方法，摒弃传统基于“点击率”、“播放量”等黑盒复合热度模型，通过自研的数学算法，从海量噪声中精准挖掘出视频的真实情绪爆点与高能名场面。

## ✨ 核心亮点

- 🚀 **自研多尺度感知增强算法**：结合微观爆发因子与宏观等分映射，彻底解决“水弹幕掩盖真实高潮”的痛点。
- 📈 **高阶非线性对比度拉伸**：采用独创的 1.8 次幂压制函数与 1:4:1 中心加权平滑，让真正的高能瞬间呈“针刺”状陡峭拔起，带来极强的视觉预警与冲击力。
- 📊 **多时间粒度聚合**：除了自研的增强分析外，依然保留了 `5s`, `10s`, `30s`, `1min`, `5min` 等传统固定切片维度的自由切换。
- 🎨 **开箱即用的沉浸式大屏**：后端 Python 一键聚合多集数据，前端 HTML 零配置读取，内置 ECharts 高能峰值排行榜与动态高亮联动。

## 🧠 核心理论模型简介

本项目的核心灵魂在于**【增强分析】**模块，其底层数学模型包含四个阶段：

1. **等分理论与时域对齐**：使用动态步长公式 $\lceil \frac{Duration}{100} \rceil$，将不同时长的视频严格归一化为 100 份。
2. **多尺度感知 (Micro-Scale Perception)**：在 1/100 宏观切片下，暗中维持 5s 的微观数据流。通过捕捉局部的瞬时峰值差异计算爆发比率，对密集区进行精准的权重补偿。
3. **底噪剥离与平滑**：减去全片最低活跃值强制归零冷场片段，并采用 `1:4:1` 滑动平滑去除生硬锯齿，最大化保留峰值的斜率。
4. **1.8 次幂的高阶非线性拉伸**：利用幂函数下凹特性，死死压制 50% 以下的干扰底噪，让 80%~100% 的名场面呈指数级放大，实现视觉上的【高能锐化】。

## 🚀 快速开始

### 1. 环境依赖
- **后端**：`Python 3.6+`（无需额外安装第三方依赖，全部采用 Python 标准库实现：`math`, `json`, `os`, `glob`, `statistics`）
- **前端**：现代浏览器（已通过 CDN 引入 `ECharts 5.5.0` 与 `MathJax`，需保持网络畅通）

### 2. 数据准备
将收集到的弹幕数据（`JSON` 格式，需包含 `progress` 毫秒字段与 `content` 字段）放入 `data/raw/` 文件夹下。
支持批量处理，例如放入：
- `人生一串S01E01.json`
- `人生一串S01E02.json`

### 3. 一键运行与分析
在终端或命令行中执行主程序：
```bash
python cli/main.py
```
执行完毕后，控制台会输出分析报告，并将多集整合数据自动输出到 `data/processed/data.js` 中。

### 4. 浏览可视化视图
直接在浏览器中双击打开项目根目录下的 `index.html` 文件。
- 左侧边栏可无缝切换不同剧集与不同时间粒度。
- 点击右侧排行榜中的高能区间，图表将会自动高亮对应的峰值范围。
